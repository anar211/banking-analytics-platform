# Data Dictionary

## Dataset

Bank Marketing Dataset (UCI Machine Learning Repository)

Total records: 45,211

---

| Column | Description |
|---------|-------------|
| age | Client age |
| job | Type of occupation |
| marital | Marital status |
| education | Education level |
| default | Has credit in default |
| balance | Average yearly account balance |
| housing | Housing loan |
| loan | Personal loan |
| contact | Contact communication type |
| day | Last contact day |
| month | Last contact month |
| duration | Last contact duration (seconds) |
| campaign | Number of contacts performed during campaign |
| pdays | Days after previous campaign |
| previous | Number of previous contacts |
| poutcome | Previous marketing campaign outcome |
| y | Client subscribed to term deposit (Target Variable) |

---

## Target Variable

**y**

- yes = Client subscribed
- no = Client did not subscribe
